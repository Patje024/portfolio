WebDevTeam


Het Web Develop Team is onderverdeeld in 5 groepen gemaakt, elk met hun eigen verantwoordelijkheden. Aan het eind van de periode zal een website worden gerealiseerd waarop de portfolio’s van de studenten te zien zijn.
Deze 5 groepen zijn:
Web Development
Project Architecten
Web Design
Test groep
Projectmanagers

Gedurende dit project ben ik zelf project manager van de project architecten. Na de verdeling is Project Management (PM) elke vrijdag met elkaar gaan zitten. Er is besproken welke stappen moeten worden gezet en wat allemaal in het plan van aanpak moet komen. Vervolgens heeft een nauwe samenwerking ervoor gezorgd dat dit project snel afgrond was. Helaas was het veel programmeren, en heb ik hier als bedrijfskundige in de architectengroep maar een kleine bijdragen aan kunnen leveren.

Verkenning / HTML

In deze workshop, verzorgd door dhr. Huijberts, is voor het eerst kennisgemaakt met HTML. In deze les is een bestaande website geinspecteerd waardoor de HTML-codering van de website zichtbaar wordt. Deze kan vervolgens worden opgeslagen op de harde schijf en worden bewerkt. Een afbeelding van mijn resultaat uit deze les volgt hieronder:

Dit was voor mij de 1e keer dat ik in aanraking ben gekomen met HTML, Ik ben erg enthousiast geworden over de minor. Grappig om te zien hoe makkelijk je hiermee bijvoorbeeld fake news kan maken.

CSS

In week 2 is door dhr. Huijberts een crashcourse HTML en CSS gegeven. HTML- Hyper Tekst Markup Language is de standaard opmaaktaal van webpagina’s. Met CSS ( Cascading Style Sheet) kan je in het HTML bestand of met een koppeling de vormgeving van de pagina aanpassen. Hieronder bijvoorbeeld Font- Family: Arial voor het lettertype.

Er is gedurende deze workshop gebruik gemaakt van visual studio code. Naast het gebruik van dit programma tijdens de lessen Smart Technology, was dit voor mij de 1e keer dat ik hiermee aan de slag ben gegaan

    • Reflectie

    Mocht ik in mijn loopbaan te maken krijgen met het ontwikkelen van software of een website weet ik welke processen doorlopen moeten worden, wie nodig zijn en hoe de communicatie verloopt achter het ontwikkelen van software/websites. Van de 5 groepen Web Development, Project Architecten, Web Design en een Test groep had ik hiervoor geen idee. Dit vervult ook een van mijn doelen en verwachtingen die ik heb opgesteld in het value proposition canvas. Voor deze minor had ik geen enkel begrip van HTML, laat staan dat er meerdere programmeertalen zijn. Het heeft mij erg getriggerd om na het afronden van bedrijfskunde nog eens stapsgewijs te leren programmeren. Na deze lessen heb ik mijn doel behaald om de basiskennis van programmeren te kennen en de begrippen die hier bijhoren. Bij HTML denk ik niet meer aan een stuurprogramma of een email, maar aan de basistekst voor het schrijven op het web, welke met CSS kan worden opgemaakt. Tot slot was het programmeren van een WeatherAPI en COVIDAPI erg leuk om te doen. De mogelijkheden met het internet zijn eindeloos. Zo hebben we bijvoorbeeld geëxperimenteerd met een dagelijkse quote toe te voegen op je website. Door het leren werken met Github, HTML, CSS en het toevoegen van API, allemaal waar ik voorheen geen enkele kennis over had, zijn de verwachtingen van de minor uit het proposition canvas voor mij zeker behaald.
